# Chat_Application

This is a simple client-server chat application using Java Sockets.

## Features

* Two-way communication
* Real-time messaging
* Chat history saved in text files

## Technologies Used

* Java
* Socket Programming
* File Handling

## How to Run

1. Compile both files:
   javac ChatServerX.java
   javac ChatClientX.java

2. Run server:
   java ChatServerX

3. Run client (in another terminal):
   java ChatClientX
